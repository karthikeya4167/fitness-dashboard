# 📊 Data Analytics Fitness Dashboard

![Python](https://img.shields.io/badge/Python-3.13-blue)
![Pandas](https://img.shields.io/badge/Pandas-Data%20Analysis-150458)
![Matplotlib](https://img.shields.io/badge/Matplotlib-Visualization-orange)

A data science script that leverages the Pandas engine to extract, transform, and load (ETL) thousands of rows of raw fitness data, converting it into a professional, visual Matplotlib dashboard. Built as part of a 20-Day Python Challenge.

## 🚀 Features
* **Automated Data Generation:** Automatically generates a robust CSV dataset of daily exercises if a local database is not detected.
* **Pandas Aggregation:** Uses advanced `groupby` and `unstack` operations to compress daily chronological data into optimized monthly summaries.
* **Matplotlib Visualization:** Renders a clean, responsive, dark-themed multi-line graph plotting volume trends over time.
